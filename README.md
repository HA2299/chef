# Chef Client - Culinary System Frontend

A modern, dynamic, and responsive user interface built with React and TypeScript, delivering a smooth experience for browsing, sharing, and generating recipes with AI.

## Key Features
* **AI Recipe Generator Interface:** Seamlessly interacts with the backend to let users input ingredients and view dynamically parsed AI-generated recipes.
* **Global State Management:** Uses Redux to efficiently handle global application state across components.
* **Component-Driven Design:** Built using reusable, modular components for clean code and high maintainability.
* **Secure Routing:** Implements protected routes to secure user-specific pages and content.

## Tech Stack
* **Core:** React, TypeScript, Vite
* **State Management:** Redux
* **Styling:** Tailwind CSS
* **HTTP Client:** Axios
* **Routing:** React Router

## Getting Started
1. Clone the repository.
2. Run `npm install` to install dependencies.
3. Configure the base API URL in your environment settings to match the backend server.
4. Run `npm run dev` to start the local development server.
